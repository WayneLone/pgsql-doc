=====================================
ABORT
=====================================

ABORT — 中止当前事务

Synopsis
-------------------------------------

.. code-block:: sql

   ABORT [ WORK | TRANSACTION ]

描述
-------------------------------------

ABORT回滚当前事务并且导致由该事务所作的所有更新被丢弃。
这个命令的行为与标准SQL命令ROLLBACK的行为一样，并且只是为了历史原因存在。

参数
-------------------------------------

:WORK:
:TRANSACTION:
   可选关键词。它们没有效果。

注解
-------------------------------------

使用 COMMIT 成功地终止一个事务。

例子
-------------------------------------

中止所有更改：

.. code-block:: sql

   ABORT;

兼容性
-------------------------------------

这个命令是一个因为历史原因而存在的 PostgreSQL 扩展。ROLLBACK 是等效的标准 SQL 命令。
