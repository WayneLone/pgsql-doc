这一章描述了 SQL 的语法。它构成了理解后续具体介绍如何使用 SQL 定义和修改数据的章节的基础。

我们同时建议已经熟悉 SQL 的用户仔细阅读本章，因为本章包含一些在 SQL 数据库中实现得不一致的以及 PostgreSQL 中特有的规则和概念。
